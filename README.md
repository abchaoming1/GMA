# GMA 2026 Sales Forecast Dashboard

This repository contains a local, editable HTML dashboard for reviewing GMA 2025 sales performance and 2026 targets.

## Files

- `index.html` - The generated dashboard. Open this file directly in a browser.
- `build_gma_dashboard.py` - The builder script used to regenerate the dashboard from the local Excel workbook.

## Notes

- The source Excel workbook is intentionally not included because it contains sales and planning data.
- The dashboard stores browser edits in `localStorage`.
- Use the in-page `Export Excel` button to export the edited dashboard data.
